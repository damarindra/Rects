Admob

Admob

Banner

In template I'm not showing the banner because I'm very dislike with banner, hehe. But you can add the banner. To add banner admob, is easy like fliping your hand. :D. here you go

  1. Add the banner key at KeyIds. (Open Loader.unity, see at hierarchy)
  2. then, add one line code at where you want to show the ads.
  3. ex: I want show it at Mainmenu, then I open script UICanvas.cs. Go to method Start and add one line code, see
AdmobController.Instance.ShowBanner();
  1. Done, you will see the banner
  2. One more, if you want to hide the banner, just call
AdmobController.Instance.HideBanner();
As Default, Banner will show at gameplay and shop

Interstitial

Okay, Interstitial is the one type ads what I like hehe. In template, I've been implements the instertitial, so you can follow this step

  1. Change the Interstitial key at KeyIds. (Open Loader.unity, see at hierarchy)
  2. Open GameManager.cs (Script)
  3. Go to method GameOver(), you will see
if (interstitialCounter >= 3)
    AdmobController.Instance.ShowInterstitial();
  1. Yeay, that is just call AdmobController.Instance.ShowInterstitial(); to show the interstitial
  2. Done. :D

Theme

Theme

Color Based

Theme with color based is just one white rectangle with color patched. We do this because we need to freeing our memory on android.

How to add more color theme?

  1. Go to loader scene. Assets/Resources/Scene/Loader.unity open it
  2. See on hierarchy, you can find GameManager. Click it
  3. see on Inspector, you can find Color Rect Array.
  4. if you want to add more theme, just change the array size.
  5. Then new Element will appear, Expand it
  6. You will see M Color, set the array size and choose the color. We recommend 6 array.
  7. Finnished, you have been add the color theme 
  8. Ouh one more, open script GameProgress.cs, go to constructor and add one / more line code (depends on how many you add theme). add this code
themeColorOpened.Add(true);
  1. parameter bool is for theme is openned or not? if not you must set the price to open it. See below to set price

Sprite Based

Theme with sprite is many prefabs with sprite renderer.

How to add more sprite theme?

  1. Open script ShapeCreator, Assets/Resources/Script/ShapeCreator.cs
  2. Then, you will see array variable MultidimensionGameObject[] gameobjectrRectangleArray. if you want add more theme, just change the size array. ex: I want 3 Sprite Based Theme, then I must change the size of array to 3 and add call the constructor multidimensionGameObject inside the array
public MultidimensionGameObject[] gameobjectRectangleArray = new MultidimensionGameObject[3] { new MultidimensionGameObject(6), new MultidimensionGameObject(6), new MultidimensionGameObject(6) };
  1. Then, on InitializeSprite() method, enter the prefabs to array that we has been create. enter the path of prefabs, start on Resources folder. ex: Resources.Load("Prefabs/Theme/Folder/SpriteName", typeof(GameObject)) as GameObject,
  1. Ouh you haven't create the prefabs? just go to unity again and see at Assets/Resources/Prefabs/Theme, just copy one folder on entire the folder. ex: copy folder Animal.
  2. Then, change the sprite in folder you have been copied. just click the prefabs and see the inspector, you can see component SpriteRenderer. change the sprite to your own sprite.
  3. Ouh one more, open script GameProgress.cs, go to constructor and add one / more line code (depends on how many you add theme). add this code
  4. parameter bool is for theme is openned or not? if not you must set the price to open it. See below to set price

Add the menu change Theme and set the price

  1. Open main.unity
  2. see at hierarchy, go to Canvas-Shop Layer-TextThemeShopcc-ScrollArea-Panel, just copy the entire of Panel, ex: copy Default, with ctrl+d
  3. Then, edit the visual with your own theme
  4. then, you've see button Buy, click it and see at Inspector.
  5. scroll down and you'll see On Click() method and 2 function inside OnClick()
  6. The first one is type of theme, ex: if Color, then set in by click it -> UICanvas -> ChooseColorTheme, if Sprite, choose UICanvas->ChooseSpriteBasedTheme.
  7. Then enter the parameter with index of theme (see bellow of the function). Be carefull to enter the index, index must available with the array of theme (step before)
  8. then see bellow, you can find UICanvas.ChooseBackgroundTheme, then enter the parameter with index of array background. 0 for white, and 1 for black.
  9. How about our save game data? is data knowing our new Theme? of course not! okay, lets fix this. Just open Script GameProgress.cs and see at constructor, add one or more line code (depend on theme addition)
themeSpriteOpened.Add(false); //Theme Based//parameter boolean for is theme openned or close.
themeColorOpened.Add(false); //Color Based//parameter boolean for is theme openned or close.
  1. Then how to set the price?
  2. Open UICanvas.cs (Script)
  3. Go to method start, add one or more line code (depend on adding theme)
goSpriteThemeButton[index].GetComponentInChildren<Text>().text = "Price";  //For Sprite Based
goColorThemeButton[index].GetComponentInChildren<Text>().text = "Price";   //For Color Based
  1. Then go back to unity, click Canvas, then scroll down. you now can see Go Sprite Theme Button or Go Color Theme Button. Drag the button buy (inside your theme) into that variable. If your theme is Color, drag it in Go Color Theme Button, otherwise.
  2. Finish
We recommend to delete save game after adding or removing theme.
We dont recommend to removing theme. Just change it. If you removing theme, just perfome to Delete Save Game. In unity, check the Delete Save Game at scene Loader, Canvas. In Android, you must delete the Data.
Background color can be found at Loader.unity and click on GameManager.
